# Features

## Main concept

The applicatiomn should accept a YouTube live stream URL and play the audio from the video. It is not necessary to play the video, only the audio. The app should be opened in the macos bar. The less additional libraries should be used, the better.

## Features

- [ ] The app shoud have two screens: main screen and settings screen. Screens should be separated by a tab view.
- [ ] Add context menu to the app icon in the macos bar. Buttons: Play, Pause, Stop, Restart, Mute, Unmute, Toggle Audio, Open in Browser, Quit
- [ ] The app should save last used YouTube live stream URL in the user defaults.
- [ ] The single language is English.
- [ ] The name of the app is "Youtube Live Player".
- [ ] The version pattern of the app is "YYYY.M.N", where N - the number of the release on month M in Year YYYY. Example: 2026.1.1 - first release in January 2026.

## Main screen

- [ ] Add a button to play the audio from the video
- [ ] Add a button to pause the audio from the video
- [ ] Add a button to stop the audio from the video
- [ ] Add a button to restart the audio from the video
- [ ] Add a button to mute the audio from the video
- [ ] Add a button to unmute the audio from the video
- [ ] Add a button to toggle the audio from the video
- [ ] Add a button to open the YouTube live stream URL in the browser
- [ ] Add a button to quit the app
- [ ] Show the current time of the live stream

## Settings screen

- [ ] Add a button to quit the app
- [ ] Show app version on the settings screen
